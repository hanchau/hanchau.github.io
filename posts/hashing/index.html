    May 10, 2020
    Concept
    Cryptography, Hash Functions, Hash Table, Hashing, Python Dictionaries

    A Brief History.

    Arousal of an idea parallely at multiple places is indicative of its need in the ecosystem. Like many ideas in Computer Science, Hashing was also introduced independently by different Scientists. According to Wikipedia, In 1953, At IBM, one computer scientist named Hans Pete Luhn wrote an internal memorandum that used Hashing (with Chaining), and about the same time a group of scientists Gene Amdahl, Elaine M. McGraw, Nathaniel Rochester, and Arthur Samuel wrote a program using hashing. After that, there has been a lot of work/research around Hashing. I can’t think of a computer science field where we don’t use Hashing. So Let’s move right on to the Concept.

    Concepts/Definitions.

    Hash Function

    The Definitions of an Hash Function is pretty straight forward - A Hash Function is a function \(\hat h\) which maps a set of arbitrary number of elements to a set of fixed number of elements i.e.

    \[ f(\xi) = \hat h(\xi) \] s.t. \( [\xi] = \infty \) and \( [\hat h(\xi)] = m \) where \( m \) is also called number of buckets. Usually \( \hat h(x) \in [0, m-1]\) .

    Collisions

    We say, A Collision has happened if, for any two inputs to the hash function \(\hat h\) we get the same output i.e.

    \[ x_1, x_2 \in \xi, x_1 \neq x_2, but, \hat h(x_1) = \hat h(x_2) \]

    Hash Table/ Hash Map

    As the name suggests, A Hash Table is a table made up of 2 primary fields, key and value. The key field stores the keys from the set \(\xi\) and the value fied stores the address of the key’s corresponding value in the memory.

    Test of a good Hash Function.

    Chi Squared Test: \( \mu(\hat h) \)

    The uniformity of the distribution of a hash function can be evaluated using the chi-squared test. Assuming we have

    \( m-buckets, n-keys, b_j \) is the number of elements in bucket j, then

    \[ \mu(\hat h) := \frac{\sum_{j=0}^{m-1} b_j(b_j+1)/2}{\frac{n}{2m}(n+2m-1)} \]

    An Example

    Let’s say we have a hash function

    \( \hat \sigma(x) \) such that \( \nexists x_1, x_2 \in \xi\) for which \( \hat \sigma(x_1) = \hat \sigma(x_2) \) \( \implies \forall j , b_j = 1 \) therefore \[ \mu(\hat \sigma) := \frac{m}{\frac{n}{2m}(n+2m-1)}\] Now if m » n \[ \mu(\hat \sigma) := \frac{m}{\frac{n}{2m}(n+2m-1)}\]
